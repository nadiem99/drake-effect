# The Drake Effect

A data essay visualizing how Drake collaborations function as cultural inflection points — launching regional artists into the mainstream and pulling entire genres into North American ears.

Single static page (`index.html`) + JSON data, hosted on GitHub Pages.

## Data sources

- **`data/artists.json`** — Google Trends search interest per artist, monthly, ±24 months from each artist's Drake collab release. Indexed so the collab month = 100. Generated by `scripts/fetch_trends.py`.
- **`data/spotify_current.json`** — current Spotify monthly listeners per artist, curated manually (shown as end-of-line annotations for scale context).

## Re-running the data pipeline

```bash
python3 -m venv .venv
.venv/bin/pip install -r scripts/requirements.txt
.venv/bin/python scripts/fetch_trends.py
```

The script is gentle with Google Trends (2s between artists, 3x retry on 429) but may still get rate-limited. Re-run if any artist fails.

To refresh current monthly listeners, edit `data/spotify_current.json` directly — each value is sourced by visiting the artist's Spotify page and copying the "X monthly listeners" figure.

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

A real HTTP server is required — opening `index.html` via `file://` will fail because the page uses `fetch()` to load JSON.

## Methodology notes

Google Trends measures **search interest**, not listeners. It's a proxy for cultural attention. The indexing (collab month = 100) means that artists with near-zero pre-collab search volume can show dramatic post-collab ratios — that amplification *is* the story the chart is telling.

Roster is split into two tiers:
- **Tier 1 — Breakouts**: limited mainstream visibility pre-collab; clear inflection at month zero.
- **Tier 2 — Already rising**: artist had prior momentum; Drake amplified rather than launched.
